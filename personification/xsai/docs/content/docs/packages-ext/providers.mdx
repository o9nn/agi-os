## Usage

### Predefined

> It reads the API Key from the environment variables via `process.env`, so it cannot be used in a browser.

```ts twoslash
import { google } from '@xsai-ext/providers' // [!code ++]
import { generateText } from '@xsai/generate-text'
import { env } from 'node:process' // [!code --]

const { text } = await generateText({
  ...google.chat('gemini-2.5-flash'), // [!code ++]
  apiKey: env.GEMINI_API_KEY!, // [!code --]
  baseURL: 'https://generativelanguage.googleapis.com/v1beta/openai/', // [!code --]
  messages: [{
    content: 'Why is the sky blue?',
    role: 'user'
  }],
  model: 'gemini-2.5-flash', // [!code --]
})
```

### Create

You can also import a create function instead of a predefined provider, making it runtime-agnostic:

```ts twoslash
// import { google } from '@xsai-ext/providers' // [!code --]
import { createGoogleGenerativeAI } from '@xsai-ext/providers/create' // [!code ++]
import { generateText } from '@xsai/generate-text'

const google = createGoogleGenerativeAI('YOUR_API_KEY_HERE') // [!code ++]

const { text } = await generateText({
  ...google.chat('gemini-2.5-flash'),
  messages: [{
    content: 'Why is the sky blue?',
    role: 'user'
  }],
})
```

### Cloud

<PkgSizeBadge pkg="@xsai-ext/providers-cloud" />

```package-install
npm i @xsai-ext/providers-cloud
```

#### Examples

##### Keys

Checkout all supported providers [here](https://github.com/moeru-ai/xsai/tree/main/packages-ext/providers-cloud/src/providers).

```ts twoslash
import * as providers from '@xsai-ext/providers-cloud'

type Keys = keyof typeof providers
```

##### OpenAI

```ts twoslash
import { createOpenAI } from '@xsai-ext/providers-cloud' // [!code ++]
import { generateText } from '@xsai/generate-text'
import { env } from 'node:process'

const openai = createOpenAI(env.OPENAI_API_KEY!) // [!code ++]

const { text } = await generateText({
  ...openai.chat('gpt-4o'), // [!code ++]
  apiKey: env.OPENAI_API_KEY!, // [!code --]
  baseURL: 'https://api.openai.com/v1/', // [!code --]
  messages: [{
    content: 'Why is the sky blue?',
    role: 'user'
  }],
  model: 'gpt-4o', // [!code --]
})
```
